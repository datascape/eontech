Site Development Tools

The following were used to create our website:

Adobe Photoshop

We combined Photoshop layers containing images of trash with 3D graphics created in SketchUp.

SketchUp

An easy way to draw in 3D. Ethan created our bots and PEM models with components assembled from the SketchUp 3D library. We also strongly recommend Tinkercad.com.

Smart Template

A simple starting point template for producing websites that scale down for mobile viewing.

Sublime Text

Sublime Text is an open source text editor for code, markup and prose.

Apache Subversion SVN

Subversion is an open source version control system that allows groups to save a history of file changes.

Entypo and Font Awesome Icons

We created our logo and icons from a collection of fonts called Entypo and Font Awesome.

iMovie

Our interview with Professor Magnus Egerstedt was edited by Ethan using the iMovie appication that ships with the Mac.

Khan Academy Scripting Lab

Our game model was scripted by Kol using the Khan Academy online scripting tutorials.
